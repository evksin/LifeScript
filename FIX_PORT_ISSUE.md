# Исправление проблемы с портом

## Проблема:
Сервер запущен на порту 3001, но вы открываете `localhost:3000/login`, поэтому получаете ошибки 404.

## Решение:

### Вариант 1: Откройте правильный URL (рекомендуется)

Откройте в браузере:
```
http://localhost:3001/login
```

### Вариант 2: Освободите порт 3000 и перезапустите сервер

1. **Остановите текущий dev-сервер:**
   - В терминале нажмите `Ctrl+C`

2. **Остановите процесс на порту 3000:**
   ```powershell
   Get-NetTCPConnection -LocalPort 3000 -ErrorAction SilentlyContinue | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ -Force }
   ```

3. **Перезапустите сервер:**
   ```powershell
   npm run dev
   ```

4. **Теперь откройте:**
   ```
   http://localhost:3000/login
   ```

## Проверка:

После открытия правильного URL (`localhost:3001/login` или `localhost:3000/login` после перезапуска):
- Страница должна загрузиться
- Должна появиться форма входа
- В консоли браузера не должно быть ошибок 404

## Если проблема сохраняется:

1. Проверьте, что сервер запущен (в терминале должно быть "Ready")
2. Проверьте URL в адресной строке браузера
3. Очистите кэш браузера (`Ctrl+Shift+R`)
